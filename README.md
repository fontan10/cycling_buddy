# Cycling Buddy

A mobile-friendly platform for children to report cycling infrastructure issues in their community — potholes, broken bike parking, unsafe routes, poor lighting, and more.

## What It Does

Cycling Buddy lets kids quickly flag problems they encounter while cycling. The experience is designed to be fast and friction-free:

1. **Landing page** — tap a category tile to select the type of issue (pothole, broken parking, unsafe route, lighting, etc.)
2. **Report form** — fill in a location, describe the problem, and optionally attach a photo
3. **Submit** — the report is logged and routed to the relevant authority

## Design Philosophy

The UI follows the "Tactile Playground" design system — an aesthetic inspired by physical wooden toys and layered paper cutouts rather than typical flat children's software. Key principles:

- Large, rounded "bubble" tap targets suited to developing motor skills
- Tonal layering (no harsh lines or dividers) to guide attention naturally
- High-contrast, accessible typography using **Plus Jakarta Sans** and **Be Vietnam Pro**
- Vibrant but grounded color palette that avoids primary-color fatigue

See [design.md](design.md) for the full design system specification.
